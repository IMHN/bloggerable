# Data messages


## data:messages.adsGoHere

Gadget type: `Global`

### Usage

```html
<data:messages.adsGoHere/>
```

```html
<b:eval expr='data:messages.adsGoHere'/>
```

```html
<b:message name='messages.adsGoHere'/>
```

| Portuguese | `Os anúncios são apresentados aqui` |
| Spanish | `Aquí van los anuncios` |
| Vietnamese | `Quảng cáo hiển thị ở đây` |
